# ArcLength

Calculate arc length for a given radius and arc measure

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'arc_length'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install arc_length

## Usage

c = ArcLength::Circle.new(5)
c.calculate_arc_length(5)
